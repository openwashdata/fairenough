# Common validators

Common validators

## Usage

``` r
validate_email(x)

validate_package_name(x)

validate_orcid(x)

validate_date(x)

validate_url(x)
```

## Value

Function that returns TRUE for valid input, FALSE otherwise

## Functions

- `validate_email()`: Email validator

- `validate_package_name()`: Package name validator

- `validate_orcid()`: ORCID validator

- `validate_date()`: Date validator (YYYY or YYYY-MM-DD)

- `validate_url()`: URL validator
