# Ensure directory exists with messaging

Ensure directory exists with messaging

## Usage

``` r
ensure_directory(
  dir_path,
  description = NULL,
  recursive = TRUE,
  verbose = TRUE
)
```

## Arguments

- dir_path:

  Path to directory

- description:

  Optional description for messages

- recursive:

  Whether to create parent directories (default TRUE)

- verbose:

  Whether to show messages (default TRUE)

## Value

The directory path
