# Read data from various sources

This function reads data from a data frame, CSV file, or Excel file. It
handles all validation and provides consistent error messages.

## Usage

``` r
read_data(data, show_messages = TRUE)
```

## Arguments

- data:

  Either a data frame or a path to a CSV/Excel file

- show_messages:

  Whether to show informational messages (default TRUE)

## Value

A data frame
